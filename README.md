## Requirements

* Python 3
* VSCode

## Setup

Define path (`workspaces_home`) where VSCode workspaces are stored. The workflow also supports subdirectories in VSCode Workspace folder

Set `workspaces_home` e.g. to /Users/user/vscode-workspaces
